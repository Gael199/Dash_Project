# 📊 Dashboard Commercial Côte d'Ivoire

Application web interactive développée avec **Python Dash** et **Plotly** pour visualiser les performances commerciales de plusieurs villes ivoiriennes.

## 🚀 Fonctionnalités

- 📈 Graphique des ventes par ville
- 💰 Répartition des profits
- 🔍 Analyse de la relation Population / Ventes / Profit
- 🗺️ Carte interactive des villes
- 🎛️ Filtrage dynamique via menu déroulant
- 📱 Interface responsive

---

## Aperçu

L'application permet de :

- Comparer les ventes entre plusieurs villes
- Analyser la rentabilité
- Visualiser les données géographiques
- Explorer les indicateurs clés de performance
